Graficos 2D

El primer dilema: SVG vs Canvas

Canvas SVG
  • API Javascript
  • Graficos de mapa de bits
  • Es veloz dibujando los graficos
  • Excelente Manipulacion de pixeles
  • Modelo de eventos basado en pixeles
  • Interacción a traves del SVG DOM API via JS
  • Graficos Vectoriales independientes de una resolución , escalables
  • Capacidades de animación incorporadas en el markup o via JS
  • Modelo de eventos basado en el objeto

El territorio de WebGL

Fuente:http://aleksandarrodic.com/p/jellyfish/ , Aleksandar Rodic

Caracteristicas de WebGL

  • API Javascript basada en OpenGL para sistemas embebidos (OpenGL ES 2.0)
  • Acelerado por hardware, permitiendo efectos mas complejos
  • Es un poco dificil de aprender
  • No soportada nativamente en IE
  • Una buena opción para manejar WebGL es Three.js

WebSockets

La solución definitiva para juegos multijugador

A diferencia de AJAX permite una conexión bidireccional natural, que no necesita de constantes 'request', ahorrando ancho de banda y permitiendo un flujo de datos en tiempo real, es ahora soportado por todos los navegadores modernos.

La Magia de Otras API's

para sacarle todo el jugo a HTML5

  • Podemos almacenar datos localmente con IndexedDB,Local Storage
  • Jugar 'off-line' mediante 'application cache'
  • Al fin juegos a pantalla completa con Fullscreen API
  • Mejor control del mouse con MouseLock API
  • Adios al WASD + ZXC con Gamepad API
  • Convirtiendo nuestras aplicaciones web en aplicaciones del SO con 'run-times' como WebRT

Ten en cuenta lo que necesitas desarrollar

You don't know the power of the Dark Side

  • La mayoria de las APIs son experimentales
  • La curva de aprendizaje puese ser larga
  • Es necesario saber como es el rendimiento de la plataforma objetivo de tu juego para saber como desarrollarlo
